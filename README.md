Alloy Game Engine
=================

A Powerful Javascript Game Engine.

Overview
=================
programming language : javascript

platform : web browse

license  : MIT License

oriented : 2D base on webgl or canvas rendering

sound effect : true

physics engine : true

integrated support tools : true
